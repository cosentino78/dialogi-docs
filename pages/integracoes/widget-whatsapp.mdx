# Widget WhatsApp

Adicione um **botao flutuante do WhatsApp** ao seu site. Com um clique, o visitante abre uma conversa direta com sua empresa no WhatsApp — sem precisar salvar o numero.

import { Callout, Steps } from 'nextra/components'
import { Screenshot } from '../../components/Screenshot'

## Como funciona

O widget e um botao verde (icone do WhatsApp) que fica no canto inferior direito do site. Quando o visitante clica:

1. Abre o WhatsApp (app ou web) automaticamente
2. O numero da empresa ja esta preenchido
3. Uma mensagem pre-configurada aparece no campo de texto
4. O visitante so precisa clicar em "Enviar"

## Personalizacao

### Alterar o numero

Troque `5511999887766` na URL `wa.me/` pelo seu numero:

```
https://wa.me/SEUNUMEROAQUI
```

### Alterar a mensagem pre-preenchida

Edite o texto apos `?text=`. Use `%20` para espacos e `%0A` para quebra de linha:

```
?text=Ola%2C%20vim%20pelo%20site!%0AGostaria%20de%20mais%20informacoes.
```

Resultado no WhatsApp:
```
Ola, vim pelo site!
Gostaria de mais informacoes.
```

### Alterar posicao

| Posicao | CSS |
|---|---|
| Canto inferior direito (padrao) | `bottom: 20px; right: 20px;` |
| Canto inferior esquerdo | `bottom: 20px; left: 20px;` |
| Mais alto | `bottom: 80px; right: 20px;` |

### Alterar tamanho

| Tamanho | CSS |
|---|---|
| Pequeno | `width: 48px; height: 48px;` |
| Medio (padrao) | `width: 60px; height: 60px;` |
| Grande | `width: 72px; height: 72px;` |

## Integracao por plataforma

### WordPress

<Steps>

### Sem plugin

1. Va em **Aparencia** → **Editor de Temas** → **footer.php**
2. Cole o codigo HTML antes do `</body>`
3. Salve

### Com plugin

Use o plugin **"Insert Headers and Footers"**:

1. Instale e ative o plugin
2. Va em **Configuracoes** → **Insert Headers and Footers**
3. Cole o codigo na secao **"Footer"**
4. Salve

</Steps>

### Wix

1. Va em **Configuracoes do site** → **Codigo personalizado**
2. Clique em **"Adicionar codigo"**
3. Cole o codigo HTML
4. Posicao: **Body - fim**
5. Aplique em: **Todas as paginas**

### Shopify

1. Va em **Loja online** → **Temas** → **Editar codigo**
2. Abra o arquivo **theme.liquid**
3. Cole o codigo antes do `</body>`
4. Salve

## Boas praticas

- **Mensagem contextual** — adapte a mensagem ao conteudo da pagina (ex: "Vi o produto X no site")
- **Horario de atendimento** — se voce nao atende 24h, informe no tooltip
- **Nao sobreponha conteudo** — certifique-se de que o botao nao cobre elementos importantes (especialmente no mobile)
- **Teste no celular** — verifique se o botao funciona em iOS e Android
